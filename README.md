# GB7 React Lab

Лабораторная работа по теме **«Цифровое представление изображения»**.
https://gb7-image-viewer.onrender.com/gb7-image-viewer/

## Что умеет приложение

- загружать изображения форматов **PNG**, **JPG/JPEG**, **GB7**;
- отображать изображение на **HTML5 canvas**;
- вписывать изображение в рабочую область холста с сохранением пропорций;
- показывать в строке состояния:
  - ширину,
  - высоту,
  - глубину цвета,
  - формат;
- отображать панель цветовых каналов:
  - `Gray` для grayscale-изображений,
  - `Gray` + `Alpha` для grayscale-изображений с маской,
  - `Red`, `Green`, `Blue` для RGB,
  - `Red`, `Green`, `Blue`, `Alpha` для RGB с прозрачностью;
- включать и выключать отдельные каналы без изменения исходного массива пикселей;
- использовать инструмент **«Пипетка»** для чтения координат пикселя, RGB и CIELAB;
- сохранять текущее изображение обратно в **PNG**, **JPG** и **GB7**;
- открывать тестовые GB7-файлы из папки `public/test-images`.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

