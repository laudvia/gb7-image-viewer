# GB7 React Lab

Лабораторная работа по теме **«Цифровое представление изображения»**.
https://gb7-image-viewer.onrender.com/gb7-image-viewer/

## Что умеет приложение

- загружать изображения форматов **PNG**, **JPG/JPEG**, **GB7**;
- отображать изображение на **HTML5 canvas**;
- показывать в строке состояния:
  - ширину,
  - высоту,
  - глубину цвета,
  - формат;
- сохранять текущее изображение обратно в **PNG**, **JPG** и **GB7**;
- открывать тестовые GB7-файлы из папки `public/test-images`.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
npm run preview
```
